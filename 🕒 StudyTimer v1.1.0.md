# 🕒 StudyTimer v1.1.0

**StudyTimer** 是一款专为学习设计的桌面时间记录与可视化工具。它不仅能帮助您精准记录每一分钟的学习时间，还能通过多维度的图表分析，助您复盘学习状态，提升专注力。

---

## ✨ 核心亮点

- **🏷️ 全新标签系统 (New!)**：
  - **活动分类**：支持为每次计时添加标签（如：学习、游戏、休息、工作）。
  - **自定义标签**：在设置中自由新增或删除标签，打造专属活动清单。
  - **堆叠可视化**：今日统计与累计分析均支持按标签堆叠显示，一眼看清时间分配。
- **智能计时与跨天处理**：点击“开始”即刻专注，点击“结束”自动记录。支持凌晨跨天记录自动拆分，确保统计数据精准到每一天。
- **今日 24 小时深度复盘**：
  - **全天堆叠分布图**：直观展示 24 小时内的活动密度，不同活动以颜色区分。
  - **分钟级下钻**：点击图表中的柱子，即可查看该小时内每 10 分钟细分段的活动堆叠分布。
  - **昨日/今日切换**：支持一键回溯昨天的活动情况。
- **多维统计分析**：支持堆叠条形图、饼图（占比）、折线图（趋势），复盘周、月、年的活动轨迹。
- **个性化壁纸与裁剪**：
  - **自适应裁剪**：优化后的裁剪器支持大图滚动，适配各种尺寸壁纸。
  - **透明度调节**：支持 0-100% 背景透明度调节，确保壁纸美观与文字清晰共存。
- **数据管理**：支持 JSON 格式的导出与导入，方便跨设备同步与备份。

## 🚀 快速开始

### 1. 环境准备
确保您的电脑已安装 Python 3.10+。

### 2. 安装依赖
在项目根目录下运行：
```bash
pip install PyQt6 matplotlib pandas numpy
```

### 3. 运行程序
```bash
python main.py
```

---

## 📦 打包为可执行程序 (.exe)

如果您想将软件打包成一个独立的小程序，放在桌面直接点击使用，请使用以下命令（需先安装 `pip install pyinstaller`）：

```bash
pyinstaller --noconsole --onefile --add-data "models.py;." --add-data "main_ui.py;." --add-data "stats_widget.py;." --add-data "settings_widget.py;." --add-data "today_widget.py;." --name "StudyTimer" main.py
```
